Sobre Mim

Olá! Sou William Kevim, bacharel em Sistemas de Informação pela Universidade Federal de Sergipe (UFS), com foco em desenvolvimento de soluções web e mobile que unem funcionalidade e design. Atuo como Desenvolvedor Web, com experiência em criação, personalização e otimização de sites, utilizando ferramentas como WordPress, Elementor, HTML5 e CSS. Meu objetivo é entregar projetos visualmente atrativos, responsivos e com bom desempenho técnico.

Sou entusiasta da área de Front-End, com conhecimentos sólidos em HTML5, CSS, e sempre buscando evoluir em JavaScript, frameworks modernos e boas práticas de UI/UX. Tenho grande interesse em Gerência de Projetos, estudando metodologias ágeis como Scrum e Kanban, e aprimorando minha visão estratégica para desenvolvimento colaborativo.

Projetos

  • Identificação de Padrões na Estimativa da Pose Humana

    O projeto teve como objetivo desenvolver um modelo de previsão para identificar padrões na estimativa da pose humana. O trabalho foi parte do meu Trabalho de Conclusão de Curso (TCC) e onde visa contribuir para a área de visão computacional e análise de movimento humano.

  • Classificação de Gatos e Cachorros com Pytorch

    O projeto teve como objetivo implementar um modelo de classificação de imagens para diferenciar entre gatos e cachorros. Utilizando a biblioteca PyTorch, foi criada uma rede neural convolucional (CNN) para realizar essa tarefa de classificação. A CNN foi treinada com um conjunto de dados contendo imagens de gatos e cachorros para aprender a reconhecer e distinguir entre as duas classes.

  • Análise Descritiva da Carteira de Trabalho e Providência Social (CTPS)

    O projeto teve como objetivo fornecer uma visão geral das informações sobre a CTPS, onde é feito uma análise descritiva destacando as variáveis do conjunto de dados, os tipos de dados identificados e algumas observações preliminares sobre as tendências encontradas.

  • Análise Descritiva de Dados de uma Usina Solar

    O projeto teve como objetivo fornecer uma visão geral das informações sobre os dados de uma usina solar, onde com base nas análises dos gráficos anuais, trimestrais, semestrais e sazonais, foi possível chegar a várias conclusões e identificar soluções para aumentar a produção de energia em épocas de baixa geração.

  • Telco Customer Churn

    O projeto teve como objetivo criar um projeto completo de ciência de dados com informações sobre clientes de uma empresa de telecomunicações, com o objetivo de prever a rotatividade (churn). Foi aplicado as técnicas tais como exploração de dados, limpeza, visualização, modelagem com regressão logística e florestas aleatórias, validação cruzada, engenharia de características e análise de impacto financeiro.

  • Back-end de Cardápio Digital

    O projeto teve como objetivo criar uma aplicação de um cardápio digital, onde foi construído com Python e FastAPI no backend.
